Automated Water Pump for Plant

This project is a smart irrigation system designed to automatically water plants based on real-time soil moisture levels, reducing the need for manual intervention and conserving water. Built using an Arduino Uno, the system uses a soil moisture sensor to detect dryness in the soil and activates a mini water pump via a relay module when the moisture level falls below a predefined threshold.

Gas Leakage Detection

This project is a safety-focused system designed to detect hazardous gas leaks using an MQ-series gas sensor. The sensor continuously monitors the air quality and alerts users through a buzzer and LED when gas concentration exceeds safe levels. An Arduino Uno processes the sensor data and triggers alarms instantly. It provides early warnings to prevent fire accidents and ensures home and industrial safety. This project deepened my skills in sensor integration, real-time monitoring, and IoT safety applications.

Arduino IDE Embedded C Arduino MQ-2 Gas Sensor

Automated Hand Sanitizer Dispenser

This project is a touchless hand sanitizer dispenser designed to promote hygiene and reduce the spread of germs. Using an Arduino Uno and an ultrasonic sensor, the system detects the presence of a hand and automatically activates a mini water pump to dispense sanitizer. The setup ensures contactless operation, making it ideal for public places, offices, and homes. It strengthened my skills in sensor integration, automation, and real-world IoT applications. The project highlights the importance of smart solutions in healthcare and public safety.

Arduino Embedded C Arduino Uno Ultrasonic Sensor (HC-SR04) DC Pump

Automated Object Detection Car using IoT

This project features a smart robotic car capable of detecting and avoiding obstacles automatically using IoT technology. Built with an Arduino Uno, ultrasonic sensors, and motor drivers, the car senses objects in its path and intelligently changes direction to avoid collisions. Real-time sensor data can be monitored or transmitted using IoT platforms for analysis and remote control. The project enhanced my skills in robotics, IoT communication, obstacle detection, and automation. It showcases how smart mobility solutions can be developed for future transportation and robotics systems.
